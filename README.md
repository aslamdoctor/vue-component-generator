# vue-component-generator
A generator for vue component

## Usage

### Install

```bash
npm install vue-component-generator -g
```

### Init Component

create a directory and run:
```bash
vcg <component-name>
```
.eg
```bash
vcg component-demo
```

### Install Depandencies

```bash
npm install
```

### Start develop

```bash
npm run dev
```

### Build Component
```bash
npm run build
```
